# tex with docker

## initialize

```shell
$ curl -fsSL https://raw.githubusercontent.com/nkjmsss/tex-docker/master/bin/install | sh
```

## How to Compile

### VSCode (suggested)

Install `LaTeX Workshop` plugin. `Paste Image` plugin might be useful too.

### docker-compose

NOTE: tex entry file is `main.tex`

```shell
$ docker-compose up
```
